# Electricity Consumption Predictions

## Project Overview

This project predicts electricity consumption using historical data and Machine Learning techniques. The model analyzes factors such as temperature, humidity, and day type to forecast future electricity usage.

## Objectives

- Analyze historical electricity consumption data
- Perform data preprocessing and feature engineering
- Build a predictive analytics model
- Forecast future electricity consumption
- Visualize trends and prediction results

## Dataset Features

- Date
- Temperature
- Humidity
- Day Type (Weekday/Weekend)
- Electricity Consumption

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Algorithm

- Linear Regression

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Model Training
5. Prediction
6. Model Evaluation
7. Data Visualization

## Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## Visualizations

- Actual vs Predicted Electricity Consumption
- Distribution of Electricity Consumption
- Temperature vs Consumption Analysis
- Humidity vs Consumption Analysis
- Correlation Heatmap

## Future Prediction

The model can predict future electricity consumption based on environmental conditions and historical patterns.
